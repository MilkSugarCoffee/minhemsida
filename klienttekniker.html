Klienttekniker

HTML HTML står för HyperText Markup Language. HTML är en av de grundläggande byggstenar ihop med CSS och Javascript.
HTML är det som utgör strukturen hos websidor. HTML är alltså ett märkspråk och inte ett programmeringsspråk, eftersom att HTML ej kan ta emot kommandon.
CSS CSS står för Cascading Style Sheets. CSS är en av de grundläggande byggstenar ihop med HTML och Javascript.
CSS gör allt som har med designen att göra. Det är med CSS som man som exempel kan göra allt från att välja typsnitt på sin text.
Javascript Javascript står för funktionaliteten. Javascript är en av de grundläggande byggstenar ihop med HTML och CSS. När man bygger en hemsida så är det alltså HTML som står för strukturen, CSS som står för designen och Javascript som står för funktionaliteten. Ett exempel på hur Javascript används är att något ändras när man klickar på en knapp. Till exempel att jag har fyllt i något i fel format. Javascript är ett programmeringsspråk.
